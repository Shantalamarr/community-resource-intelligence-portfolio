# Apex Retail Inventory Analysis

## Overview

Designed and queried a relational inventory database using MySQL Workbench.

The project demonstrates SQL skills including database creation, table creation, joins, aggregation, and business analysis.

---

## Business Problem

Apex Retail needs to analyze inventory levels, supplier performance, inventory value, and products requiring restocking.

## Business Questions Answered

- How many products are in inventory?
- What is the total inventory value?
- Which category has the highest inventory value?
- Which products require restocking?
- Which supplier provides the most products?
- Which supplier has the highest rating?
- Which state has the most suppliers?
